# Algorithm Visualizer Demo

Real-time visualization of classic sorting algorithms.

## Features

- Visual representation of sorting process
- Multiple algorithms: Bubble Sort, Selection Sort, Insertion Sort
- Adjustable array size and animation speed
- Color-coded element comparisons
- Interactive controls

## Usage

Run directly:
```bash
python algorithm_visualizer.py
```

Or launch from the main launcher:
```bash
cd ../..
python launcher.py
```

## Controls

- **Array Size Slider**: Change number of elements (10-200)
- **Speed Slider**: Control animation speed (0.001-0.1)
- **Algorithm Radio Buttons**: Select sorting algorithm
- **Start Button**: Begin sorting visualization
- **Reset Button**: Generate new random array

## Algorithms

- **Bubble Sort**: O(n²) - Repeatedly swaps adjacent elements
- **Selection Sort**: O(n²) - Finds minimum element and places it at beginning
- **Insertion Sort**: O(n²) - Builds sorted array one element at a time
