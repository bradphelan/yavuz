# Parametric 3D Curves Demo

Explore beautiful 3D mathematical curves with interactive parameter controls.

## Features

- Multiple parametric curve types
- Three adjustable parameters
- Color gradient visualization
- Configurable point density
- Interactive 3D rotation

## Usage

Run directly:
```bash
python parametric_3d.py
```

Or launch from the main launcher:
```bash
cd ../..
python launcher.py
```

## Controls

- **Param A/B/C Sliders**: Adjust curve parameters (0.1-5.0)
- **Curve Type Radio Buttons**: Select curve type
- **Points Text Box**: Set number of sample points
- **Mouse**: Rotate and zoom the 3D view

## Curve Types

- **Helix**: Spiral curve around a cylinder
- **Torus Knot**: Knot that lies on the surface of a torus
- **Lissajous**: 3D Lissajous curves with three frequencies
- **Spiral**: Spherical spiral curve
