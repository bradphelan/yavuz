# Strange Attractor Explorer

Animate classic strange attractors with interactive controls and mouse-driven positioning.

## Features

- Lorenz and Rossler attractors
- Live animation with adjustable speed
- Mouse drag to move the attractor locus
- Parameter sliders for each system
- Trail length control, pause, and reset

## Usage

Run directly:
```bash
python strange_attractor.py
```

Or launch from the main launcher:
```bash
cd ../..
python launcher.py
```

## Controls

- **System Radio Buttons**: Switch between Lorenz and Rossler
- **Parameter Sliders**: Tune system parameters
- **Time Step**: Integration step size
- **Steps/Frame**: Simulation speed
- **Trail**: Number of points in the trajectory
- **Pause / Reset**: Stop or restart the animation
- **Mouse Drag on Plot**: Move the attractor locus
